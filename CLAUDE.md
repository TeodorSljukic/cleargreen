# ClearGreen

## About
Digital marketing agency website with ACF blocks and Gatekeeper landing page system.

## Tech Stack
- WordPress theme (PHP)
- Local by Flywheel for development
- ACF Pro with custom blocks
- Bootstrap 5
- GSAP + Swiper.js
- Customizer controls for Swiper logo settings

## Theme Structure
- `functions.php` - ACF blocks, Gatekeeper landing page templates, Customizer Swiper controls
- Two Gatekeeper landing page templates (Standard + Titan)
- ACF block templates

## Plugins
- ACF Pro
- Elementor
- Rank Math SEO
- LiteSpeed Cache
- Wordfence Security
- Google Site Kit
- Contact Form 7
- All-In-One WP Migration
- (9 total)

## Development Notes
- **ACF Blocks** - Custom Gutenberg blocks
- **Gatekeeper Landing Pages** - Two distinct landing page templates:
  - Standard: Regular marketing landing page
  - Titan: Enhanced version with separate GSAP/Swiper/Bootstrap JS stack
- **Customizer Swiper Controls** - Logo carousel settings via WP Customizer
- Completely separate JS stack for Titan landing pages
